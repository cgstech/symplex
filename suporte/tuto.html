Tutorial do Sistema de Gerenciamento - CGS Symplex

Este tutorial orienta sobre como utilizar o sistema de gerenciamento de pedidos e a base de conhecimento.

1. Introdução

Bem-vindo ao sistema de gerenciamento de pedidos da CGS Symplex. Este sistema foi projetado para facilitar o gerenciamento de pedidos e a consulta a informações relevantes na base de conhecimento.

2. Acessando o Sistema

Para acessar o sistema, você deve inserir suas credenciais de login na página de acesso. Após o login, você será redirecionado para o painel de controle do sistema.

3. Navegando pela Interface

A interface do sistema é intuitiva e fácil de usar. Utilize a barra de navegação superior para acessar as diferentes seções, como "Dashboard", "Abrir Chamado" e "Base de Conhecimento".

4. Adicionando Conteúdo à Base de Conhecimento

4.1 Formulário de Adição

Para adicionar um novo artigo ou tutorial à base de conhecimento, navegue até a seção correspondente e clique no botão "Adicionar Novo Conteúdo".

4.2 Campos do Formulário

  • Título: Insira um título claro e descritivo.
  • Descrição: Forneça uma descrição detalhada do conteúdo.
  • Data de Publicação: Insira a data em que o conteúdo foi criado.

4.3 Salvar e Visualizar

Após preencher todos os campos, clique no botão "Salvar" para adicionar o conteúdo à base de conhecimento. Você poderá visualizar o artigo na seção de artigos recentes.

5. Buscando Informações na Base de Conhecimento

Utilize a barra de pesquisa na parte superior da base de conhecimento para encontrar informações específicas. Digite palavras-chave relacionadas ao que você está buscando.

6. Visualizando Artigos e Tutoriais

Os artigos e tutoriais são exibidos em formato de cards. Clique em um card para abrir a visualização completa do conteúdo.

7. Considerações Finais

O sistema de gerenciamento de pedidos da CGS Symplex foi desenvolvido para ser simples e eficaz. Se você tiver dúvidas ou sugestões, não hesite em entrar em contato com o suporte.
